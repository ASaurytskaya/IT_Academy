Пакет loops содержит задания по циклам (1.1 - 1.6), пакет arrays содержит задания по массивам(2.1 - 2.4), пакет sorts содержит задания по сортировкам (3, 4.1 - 4.2), пакет utils содержит вспомогательные классы.
В общем случае, цифры в конце имени класса, начинающегося с "Main", соответствуют номеру задачи (пункту и подпункту домашнего задания). Кроме того, заданию 2.2 соответствует класс ArraysIteration, заданию 3 - файлы shaker_diagram.vsdx и shaker_diagram.svg, заданию 4 - классы SortsMain и SortsUtils.
Дополнительно: класс NumberEntry содержит методы, проверяющие корректность ввода чисел через консоль.
